## Coding 2 Final Project

For your final project, you will develop a Python-based command-line interface (CLI). Your program must prompt the user for a specific data request via the terminal, fetch the corresponding information from an external API, and display the processed results back to the terminal. Additionally, all data retrieved from the API must be structured and stored within a local data file (such as JSON or CSV) or a defined data structure for persistence.

### Project Requirements
Your program must have the following components
- must have at least 1 user defined function
- must take in a user input via the input function
- must use atleast 1 API and request atleast 1 endpoint
- must use the request module
- must use sqlite for the database
- must be able to save requested API data to the database
- must be able to display the data in the terminal
